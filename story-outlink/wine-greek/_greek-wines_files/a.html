                      Yeap, set operations can be very helpful in Python because a) the underlying set implementation (and dict's for that matter) is very fast and b) their expression mechanics are very powerful. e.g. set comprehensions do mimic the corresponding mathematical notation. For example python's {2*x for x in {1,2,3} } is pretty close to the mathematical {2x: x∈{1,2,3} }
